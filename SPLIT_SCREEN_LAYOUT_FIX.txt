================================================================================
SPLIT-SCREEN LAYOUT FIX - Daily View Article Detail
================================================================================

PROBLEM
-------
In Daily View article detail mode, Summary and Original panes were stacking
vertically instead of displaying side by side.

CAUSE
-----
- Parent container had flexWrap: "wrap" 
- Each child had minWidth: "400px"
- When total width < 800px, panes would wrap and stack

SOLUTION
--------
Changed flex layout to force side-by-side display:

1. Parent Container (lines ~6110-6115):
   - Added: flexDirection: "row"
   - Added: width: "100%"
   - Changed: gap from "10px" to "16px"
   - Removed: flexWrap: "wrap"

2. Child Panes (lines ~6088-6108):
   - Changed: flex from "0 0 45%" / "0 0 55%" to "1"
   - Changed: minWidth from "400px" to "0"

3. PDF Iframe (lines ~6002-6009):
   - Changed: height from "calc(100vh - 240px)" to "75vh"
   - Added: display: "block"
   - Kept: width: "100%"

================================================================================
CHANGES MADE
================================================================================

File: twifo.py

Location: _build_split_screen_layout() function (lines ~6085-6115)

BEFORE:
-------
Parent style:
  display: flex
  gap: 10px
  flexWrap: wrap  <-- Caused stacking

Child style:
  flex: 0 0 45% / 0 0 55%  <-- Fixed widths
  minWidth: 400px  <-- Triggered wrapping

Iframe style:
  height: calc(100vh - 240px)  <-- Too tall

AFTER:
------
Parent style:
  display: flex
  flexDirection: row  <-- NEW: Force row layout
  gap: 16px  <-- Increased from 10px
  width: 100%  <-- NEW: Full width

Child style:
  flex: 1  <-- Equal flex grow
  minWidth: 0  <-- Allow shrinking

Iframe style:
  width: 100%
  height: 75vh  <-- Constrained height
  display: block  <-- NEW: Block display

================================================================================
RESULT
================================================================================

✓ Summary and Original panes now display side by side
✓ Each pane takes 50% width (flex: 1 on both)
✓ Panes can shrink below 400px if needed
✓ PDF iframe constrained to 75vh (prevents overflow)
✓ No stacking on normal desktop screens
✓ Daily Summary page unchanged (different code path)
