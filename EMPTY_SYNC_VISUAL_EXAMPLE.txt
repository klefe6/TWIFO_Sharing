KEY IMPROVEMENTS
================================================================================

1. SYNCHRONIZATION
   - Both panels update simultaneously when date changes
   - No stale content ever visible

2. CONSISTENCY
   - Both panels show the same date: "December 31, 2026"
   - Both panels use the same format: "%B %d, %Y"

3. CLARITY
   - Empty state is obvious and centered
   - Helpful message guides user to select different date
   - No confusion about which date is being viewed

4. PERFORMANCE
   - Right panel clears immediately (no delay)
   - No unnecessary rendering when artifacts exist
   - Early return prevents wasted computation

================================================================================

    |
    v

    |
    +-> Loads artifacts for new date
    +-> Updates daily-articles-store to [] (empty)
    +-> Updates left panel to show "No articles found"
    |
    v
display_daily_article_summary callback fires (NEW BEHAVIOR)
    |
    +-> Triggered by: daily-articles-store update (not button click)
    +-> Checks: artifacts is empty
    +-> Action: Clear right panel with empty state
    +-> Shows: "No Articles Found" with correct date
    |
    v
RESULT: Both panels synchronized and showing empty state
